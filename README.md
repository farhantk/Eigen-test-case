# Eigen Test-case

## Tech Stack
- NestJS
- MySQL
- Prisma

## Installation

```sh
git clone https://github.com/farhantk/Eigen-test-case.git
cd Eigen-test-case
npm install
npx prisma migrate dev
npm run seed
npm run dev 
```

## Unit test

```sh
npm run book
npm run member
npm run borrow
```

## Swagger

```sh
http://localhost:3000/api
```
