# Eigen Test-case

## Tech Stack
- NestJS
- MySQL
- Prisma

## Installation

```sh
git clone https://github.com/farhantk/DOT-Intern.git
cd DOT-Intern
npm install
npx prisma migrate dev
npm run seed
npm run dev 
```

## Unit test

```sh
npm run book
npm run member
npm run borrow
```
